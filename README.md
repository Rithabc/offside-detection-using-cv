# Offside Detection System using Computer Vision

This project implements an **Offside Detection System in Football (Soccer)** using Computer Vision techniques.

## Description

This system analyzes football match images/videos to detect offside situations by tracking player positions and applying geometric rules.

The project uses Computer Vision techniques to:
- Detect players on the field
- Identify team positions
- Estimate offside situations

## How to Run

To run the project, simply execute the main pipeline file:

```bash
python main_pipeline.py
